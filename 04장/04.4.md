# 04.4 파워포인트에서 클로드 사용하기

## 바로 37 슬라이드 마스터 만들기 — 01단계

```
다음 요구사항을 반영한 슬라이드 마스터와 샘플 슬라이드를 생성해.
요약 :
최상위 전략 컨설팅사 스타일의 고밀도 전문 슬라이드 디자인.
레이아웃 :
정밀한 차트(누적 막대, 워터폴, 선그래프)와 상세 데이터 표 포함
얇고 깔끔한 선으로 구성된 전략 다이어그램 또는 2x2 매트릭스
다단 구성의 정보 고밀도 레이아웃
스타일 :
미니멀하지만 정보 중심의 깔끔하고 권위 있는 분위기
헤드라인은 세리프, 데이터·차트는 산세리프 폰트
화이트 배경, 블랙 텍스트, 딥블루와 그레이 톤 활용
표는 얇은 테두리, 그래프는 정밀한 선 사용
```

## 바로 38 발표 자료 만들기 — 01단계

```
'AI 시대의 마케팅 전략'이라는 주제로 10분 발표용 슬라이드를 구성해줘. 8장 정도로 만들어줘.
```

## 바로 38 발표 자료 만들기 — 03단계

```
전체 슬라이드의 디자인을 통일해줘. 깔끔한 비즈니스 스타일로, 주 색상은 남색과 흰색으로 해줘.
```

## 바로 38 발표 자료 만들기 — 04단계

```
3번 슬라이드에 AI 마케팅 시장 규모를 선그래프로 그려줘. 2022년부터 2026년까지 연도별로.
```

## 바로 38 발표 자료 만들기 — 05단계

```
모든 슬라이드에 발표자 노트를 추가해줘. 각 슬라이드당 1분 정도 말할 수 있는 분량으로, 자연스러운 화법으로 써줘.
```

## 바로 38 발표 자료 만들기 — 06단계

```
이 슬라이드를 검토하고 개선할 점을 알려줘. 텍스트가 너무 많은 슬라이드는 핵심만 남기고 줄여줘.
```
