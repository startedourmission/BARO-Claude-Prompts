# 06.3 PDF 문서 편집하기

## 바로 50 PDF에서 표 추출해 엑셀로 만들기 — 01단계

```
'월별보고서.pdf' 파일에서 모든 표를 추출해 '월별보고서+표.xlsx' 파일로 만들어줘.
```

## 바로 50 PDF에서 표 추출해 엑셀로 만들기 — 02단계

```
'재무제표.pdf' 파일의 1페이지부터 2페이지까지 있는 표를 하나의 연속된 표로 추출해 엑셀로 만들어줘.
```

## 바로 50 PDF에서 표 추출해 엑셀로 만들기 — 03단계

```
'판매데이터.pdf'의 표를 추출해서 다음과 같이 처리해줘.
1. 천 단위 구분 쉼표 제거
2. 금액에서 '원' 문자 제거
4. 엑셀 파일로 저장
```

## 바로 51 여러 개의 PDF를 하나로 합치기 — 01단계

```
'보고서' 폴더에 있는 모든 PDF 파일을 하나로 합쳐줘. 파일 이름 순서대로 합치고, '통합보고서.pdf'로 저장해줘.
```

## 바로 51 여러 개의 PDF를 하나로 합치기 — 02단계

```
다음 순서로 PDF를 합쳐줘.
1. '프로젝트현황보고서.pdf' 전체
2. '시장조사보고서.pdf'의 1~2페이지
결과 파일 : '최종문서.pdf'
```

## 바로 52 PDF에 내용 입력하기 — 01단계

```
'신청서_양식.pdf'의 폼 필드를 다음 정보로 채워줘.
- 이름 : 김서연
- 생년월일 : 1990-05-15
- 연락처 : 010-1234-5678
- 주소 : 서울시 강남구 테헤란로 123
채워진 파일을 '신청서_김서연.pdf'으로 저장해줘.
```

## 바로 52 PDF에 내용 입력하기 — 02단계

```
'신청자목록.xlsx'의 각 행 데이터로 '신청서_양식.pdf'를 채워줘. 파일명은 '신청서_이름.pdf'로 저장해줘.
```
