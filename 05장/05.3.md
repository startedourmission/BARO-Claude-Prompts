# 05.3 필요한 스킬 직접 만들기

## 바로 45 뉴스 브리핑 스킬 만들기 — 02단계

```
스킬 크리에이터 스킬을 사용해서 AI 기술 뉴스 브리핑 스킬을 만들어줘. 결과물은 마크다운 파일로 저장해. 다운로드할 수 있게 하나의 압축 파일로 줘.
```

## 바로 45 뉴스 브리핑 스킬 만들기 — 04단계

```
뉴스 브리핑 스킬을 사용해 오늘의 뉴스를 요약해줘.
```

## 바로 46 기존 스킬 최적화하기 — 01단계

```
/skill-creator Project-planner 스킬이랑 creating-financial-models 스킬을 합쳐서 하나로 만들어줘. 프로젝트 기획서를 작성하면서 동시에 수익성 분석 엑셀까지 생성하는 스킬이야. 스킬 이름은 project-analyzer로 해줘.
```

## 바로 46 기존 스킬 최적화하기 — 02단계

```
project-analyzer 스킬을 다음과 같이 수정해줘.
- 기획서 템플릿에 "리스크 분석" 섹션을 추가해줘.
- 수익성 분석에서 손익분기점을 자동으로 계산하도록 해줘.
- 결과물은 마크다운 기획서 + 엑셀 재무 모델, 두 파일로 출력해줘.
```

## 바로 46 기존 스킬 최적화하기 — 03단계

```
/project-analyzer 사내 카페 운영 프로젝트를 분석해줘. 초기 투자 5천만 원, 월 운영비 800만 원, 예상 일평균 고객 120명이야.
```

## 바로 활용 회사 맞춤형 회의록 자동 정리 스킬 만들기 — ??단계

```
스킬 크리에이터 스킬을 사용해서 ‘회의록 자동 정리 스킬’을 만들어줘. 첨부한 우리 회사의 회의록 양식(참석자, 핵심 안건, 결정 사항, 담당자별 액션 아이템)을 기본으로 삼고, 입력된 대화에서 이 양식에 맞춰 내용을 추출하라는 지침을 넣어줘. 결과물은 다운로드할 수 있게 하나의 압축 파일로 줘.
```
