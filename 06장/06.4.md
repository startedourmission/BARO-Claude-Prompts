## 바로 54 보고서 초안 자동 작성하기 — 01단계

```
report 템플릿으로 보고서를 만들어줘.
- 제목 : 2024년 상반기 마케팅 성과 보고
- 작성자 : 김서연
- 작성일 : 오늘 날짜
- 내용 : 1분기 매출이 전년 대비 15% 증가했고, SNS 마케팅 캠페인의 전환율이 3.2%를 달성했다는 내용으로 작성해줘.
- 파일명 : '마케팅_성과보고.hwpx'
```

## 바로 55 스킬 수정해서 우리 회사에 맞추기 — 01단계

```
hwpx 스킬을 수정해서 문서 위쪽에 항상 우리 팀 로고 이미지와 팀 이름인 전략기획팀이 들어가도록 해.
본문 글꼴은 함초롱바탕 10pt, 줄간격 160%
항목 체계는 다음과 같이 설정해줘

첫째 항목 : 1., 2., 3. ...
둘째 항목 : 가., 나., 다. ...
셋째 항목 : 1), 2), 3) ...
넷째 항목 : 가), 나), 다) ...
다섯째 항목 : (1), (2), (3) ...
여섯째 항목 : (가), (나), (다) ...
```

## 바로 55 스킬 수정해서 우리 회사에 맞추기 — 02단계

```
수정된 스킬로 테스트 문서를 만들어줘. 제목은 3월 업무 보고, 내용은 아무거나 짧게 채워줘.
```
