## 바로 27 지메일 연결해 메일 정리하고 답장 쓰기 — 05단계

```
받은 메일 중에서 읽지 않은 메일을 모두 확인해줘. 각 메일의 발신자, 제목, 중요도를 정리해서 표로 보여줘.
```

## 바로 27 지메일 연결해 메일 정리하고 답장 쓰기 — 07단계

```
읽지 않은 메일을 업무 메일, 뉴스레터, 광고, 개인 메일로 분류해줘. 각 카테고리별로 몇 통인지도 알려줘.
```

## 바로 27 지메일 연결해 메일 정리하고 답장 쓰기 — 08단계

```
김 팀장님 메일에 답장을 써줘. 내일 오전 10시 회의에 참석할 수 있다고 전하고, 안건에 대한 내 의견은 회의 전에 문서로 공유하겠다고 적어줘.
```
