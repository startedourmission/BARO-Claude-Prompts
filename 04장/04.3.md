# 04.3 엑셀에서 클로드 사용하기

## 바로 34 엑셀에 클로드 추가 기능 설치하기 — 05단계

```
가상의 쇼핑몰 회원 데이터 샘플을 100건 만들어줘.
```

## 바로 35 조건부 서식 자동으로 만들기 — 01단계

```
최근 접속일이 2021년인 고객의 행은 배경색을 분홍색으로 바꿔줘.
```

## 바로 35 조건부 서식 자동으로 만들기 — 02단계

```
총 구매 금액이 100만 원 이상인 고객의 행은 글자 색을 파란색으로 표시하는 조건부 서식을 만들어줘.
```

## 바로 35 조건부 서식 자동으로 만들기 — 03단계

```
색상 스케일, 데이터바, 아이콘 집합, 수식 기반 조건부 서식을 한 번에 만들어서 적용해줘. (구매 금액 히트맵 + 등급별 색상 등)
```

## 바로 36 데이터 분석하고 대시보드 만들기 — 01단계

```
새 시트에 고객 RFM 분석해서 정리해줘.
```

## 바로 36 데이터 분석하고 대시보드 만들기 — 02단계

```
새 시트에 대시보드를 만들어줘. 지역이랑 회원 등급을 드롭다운으로 선택할 수 있게 하고, 선택에 따라 고객 수 / 총 구매 금액 / 평균 구매 횟수 / 마케팅 동의율 KPI가 자동으로 바뀌게 해줘.
```
