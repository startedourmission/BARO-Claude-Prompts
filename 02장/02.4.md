# 02.4 비주얼라이저로 작업 시각화하기

## 바로 16 비주얼라이저로 차트 그리기 — 01단계

```
내 월급 350만 원의 지출 구조를 도넛 차트로 시각화해줘. 상세 항목은 가상의 데이터를 사용해.
```

## 바로 16 비주얼라이저로 차트 그리기 — 03단계

```
최근 6개월 지출 추이를 꺾은선 차트로 만들어줘. 상세 항목은 가상의 데이터를 사용해.
```

## 바로 16 비주얼라이저로 차트 그리기 — 04단계

```
우리 집 재정 건강 점수를 레이더 차트로 만들어줘. 슬라이더로 카테고리 별 지출을 조정하면 차트에도 반영되게 반들어줘. 그리고 이상적인 재정 건강 차트를 겹치게 그려줘.
```
